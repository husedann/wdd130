Site Name

MyMemories

Site Purpose with Goals

Easiest way to document your family history one memory at a time

IRemember method will contain a text input box that will allow the input of of each memory as it pops into the mind of the user. The memory will be stored in a database and will be able to be viewed and edited, since more of the memory might surface.

Prompted Memory method will allow the user to look at a list of memory questions that the user can input into the IRemember section.

Timeline Entry method will capture a date, place, and the event. Then the user will be able to input the memory of that event into the IRemember section.

Target Audience

MyMemories site will be a place to gather the following:

  • My Memories
  • Gallery
  • My Timeline
